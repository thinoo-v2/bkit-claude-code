# Starter 프로젝트 초기화

## 수행 작업

1. **프로젝트 구조 확인**
   - package.json 또는 index.html 확인
   - 기존 docs/ 폴더 확인

2. **PDCA 문서 폴더 생성**
   ```
   docs/
   ├── 01-plan/
   │   ├── _INDEX.md
   │   └── features/
   ├── 02-design/
   │   ├── _INDEX.md
   │   └── features/
   ├── 03-analysis/
   │   └── _INDEX.md
   └── 04-report/
       └── _INDEX.md
   ```

3. **기본 문서 생성**
   - 각 폴더에 _INDEX.md 생성
   - 프로젝트 README.md 업데이트 (필요시)

4. **완료 메시지**
   - 생성된 구조 표시
   - 다음 단계 안내

## 실행 조건

- Starter 레벨에 적합한 프로젝트인지 확인
- 이미 초기화된 경우 건너뛰기 (덮어쓰지 않음)

## 다음 단계 안내

```
✅ Starter 프로젝트가 초기화되었습니다!

다음 단계:
1. /pdca-plan [기능명] - 첫 번째 기능 계획 작성
2. 또는 기능 구현을 요청하면 자동으로 PDCA가 적용됩니다
```
